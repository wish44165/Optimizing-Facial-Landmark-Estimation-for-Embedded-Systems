<summary>0.1.Create Conda Environments</summary>

```bash
$ conda create -n yolov8 python=3.10 -y
$ conda activate yolov8
$ pip install ultralytics
$ git clone https://github.com/ultralytics/ultralytics.git
$ cd ultralytics/
```
  
</details>
